# 🚗 Smart Parking System

Привіт! Це застосунок для **смарт-паркінгу**, який допомагає оператору швидко бачити, що відбувається на парковці, і керувати системою без зайвого хаосу.  
Працює як **зручна панель керування** для Windows та Android і спілкується з Arduino-контролером.

## ✨ Що ця апка допомагає робити

- дивитися, які місця **вільні**, **зайняті** або **вимкнені**
- керувати **воротами**
- міняти **налаштування контролера**
- редагувати **тексти на дисплеї**
- працювати з **RFID-картками**
- дивитися **події системи**
- мати все в одному спокійному інтерфейсі, а не в купі сирих кнопок

## 📱 Основні екрани

- `Connection` — підключення до контролера
- `Dashboard` — швидкий огляд системи
- `Parking` — карта паркомісць
- `Gate` — керування воротами
- `Activity` — недавні події
- `Admin` — розширені операторські дії
- `Settings` — середовище, мова, службові налаштування

## 🤖 Команди до Arduino

Нижче коротка таблиця команд, які використовує застосунок для роботи з контролером.

| Команда | Що робить |
|---|---|
| `HELLO SPS` | перевіряє, що це саме наш контролер |
| `GET PROFILE` | просить профіль пристрою |
| `GET CONFIG` | просить поточну конфігурацію |
| `GET SNAPSHOT` | просить поточний знімок стану |
| `GATE FORCE_OPEN ON` | вмикає примусово відкриті ворота |
| `GATE FORCE_OPEN OFF` | вимикає примусово відкриті ворота |
| `GATE OPEN_TEMP` | тимчасово відкриває ворота |
| `GATE CLOSE` | закриває ворота |
| `GATE LOCK ON` | блокує ворота |
| `GATE LOCK OFF` | розблоковує ворота |
| `CONFIG SAVE` | зберігає конфігурацію |
| `CONFIG RESET` | скидає конфігурацію до стандартної |
| `CONFIG OPEN_ANGLE <value>` | ставить кут відкриття воріт |
| `CONFIG CLOSED_ANGLE <value>` | ставить кут закриття воріт |
| `CONFIG OPEN_DURATION_MS <value>` | ставить тривалість відкриття |
| `CONFIG THRESHOLD_CM <value>` | ставить поріг датчика відстані |
| `CONFIG TELEMETRY_MS <value>` | ставить інтервал телеметрії |
| `PARKING ENABLE <slot>` | вмикає паркомісце |
| `PARKING DISABLE <slot>` | вимикає паркомісце |
| `CARDS ALLOWED ADD <uid>` | додає дозволену картку |
| `CARDS ALLOWED REMOVE <uid>` | видаляє дозволену картку |
| `CARDS ALLOWED CLEAR` | очищає список дозволених карток |
| `CARDS BLOCKED ADD <uid>` | додає заблоковану картку |
| `CARDS BLOCKED REMOVE <uid>` | видаляє заблоковану картку |
| `CARDS BLOCKED CLEAR` | очищає список заблокованих карток |
| `DISPLAY FORCE ON` | вмикає примусовий текст дисплея |
| `DISPLAY FORCE OFF` | вимикає примусовий текст дисплея |
| `DISPLAY TEXT FORCED <text>` | задає примусовий текст |
| `DISPLAY TEXT DEFAULT <text>` | задає стандартний текст |
| `DISPLAY TEXT ALLOWED <text>` | задає текст для дозволеної картки |
| `DISPLAY TEXT BLOCKED <text>` | задає текст для заблокованої картки |
| `DISPLAY TEXT INVALID <text>` | задає текст для невірної картки |
| `DISPLAY TEXT LOCKED <text>` | задає текст для заблокованих воріт |

## ❤️ Для чого цей проєкт

Головна ідея проста:

> зробити паркінг не “набором сирих кнопок”, а зрозумілою і зручною системою керування.

Тобто ця апка потрібна, щоб:
- оператору було легше працювати
- стан паркінгу був видимий одразу
- керування воротами і картками було акуратним
- систему можна було розвивати далі без повного переписування
